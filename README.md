## Deskripsi

Project-December adalah platform web musiman yang hanya aktif penuh di bulan Desember.  
Pengguna bisa menulis pengakuan, pengalaman, atau cerita anonim, yang kemudian muncul sebagai animasi **butiran salju**. Klik salju untuk membaca isi lengkap tulisan.
